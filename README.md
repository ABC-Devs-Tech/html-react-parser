# html-react-parser

An HTML to React parser.

```js
var Parser = require('html-react-parser');
var reactElement = Parser('<p>Hello, world!</p>');

// ReactDOM.render(reactElement, node);
```

## Installation

```sh
$ npm install html-react-parser
```

## Testing

```sh
$ npm test
```
